在父目录根节点：git submodule add （git仓库地址） （本地仓库路径）：例如
    1、git submodule add https://github.com/llhere/SpringCliud-Config-Center.git chen-cloud-mashibing/SpringCliud-Config-Center
    2、git status
    3、git commit -m "add submodule chen-cloud-mashibing/SpringCliud-Config-Center"
    4、向远程仓库推送

遇到错误删除文件：
    1、git rm -f EurekaColony
    2、git status
    3、git commit -m "remove"
    4、向远程仓库推送
